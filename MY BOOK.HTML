      1. STUDENT TABLE:

        2. The Student table stores information about individual students enrolled in the educational institution. The Typical attributes might include student ID, name, email, contact number, address, and any other relevant demographic information. This table is essential for maintaining a comprehensive database of students enrolled in courses and tracking their attendance records.


      2. COURSE TABLE:

        3. The Course table stores information about the courses offered by the educational institution. The Typical attributes might include course ID, course name, course code, description, instructor ID (linked to the Faculty table), and any other relevant details about the course. This table helps administrators and instructors manage course offerings, assign instructors, and track course-related data such as attendance, grades, and schedules.


      3. ATTENDANCE TABLE:

        4. The Attendance table records the attendance data for students in each course session. The Typical attributes might include attendance ID, student ID (linked to the Student table), course ID (linked to the Course table), date, and attendance status (e.g., present, absent, late). This table is crucial for tracking student attendance over time, generating attendance reports, and identifying patterns or trends in student attendance behavior.


      4. FACULTY TABLE:

        5. The Faculty table stores information about instructors or faculty members teaching the courses. The Typical attributes might include faculty ID, name, email, contact number, department, and any other relevant details about the faculty member. This table facilitates the assignment of instructors to courses, tracking instructor information, and managing faculty-related data within the system.


      5. ENROLLMENTS TABLE:

        6. The Enrollments table records the enrollment status of students in each course. The Typical attributes might include enrollment ID, student ID (linked to the Student table), course ID (linked to the Course table), enrollment date, and any other relevant enrollment-related details. This table helps administrators manage student enrollments in courses, track enrollment history, and ensure accurate course rosters.
